# Syntecxhub_Expensive_Tracker_CLI
Overview
Expense Tracker CLI is a command-line application developed using Python that helps users manage their income and expenses. The application stores financial records in a CSV file and generates monthly summaries to help users track their spending and savings.
Features
- Add income and expense entries
- Store records in CSV format
- View all transactions
- Generate monthly financial summaries
- Calculate total income, expenses, and balance
- Simple command-line interface
Technologies Used
- Python
- CSV Module
- Datetime Module
Project Structure
ExpenseTracker/
│
├── expense_tracker.py
├── expenses.csv
└── README.md
How to Run
1. Install Python 3.
2. Save the code as "expense_tracker.py".
3. Open Terminal or Command Prompt.
4. Navigate to the project folder.
5. Run:
python expense_tracker.py
Usage
1. Select "Add Entry" to record income or expenses.
2. Select "View Entries" to display all records.
3. Select "Monthly Summary" to view monthly financial statistics.
4. Select "Exit" to close the application.
